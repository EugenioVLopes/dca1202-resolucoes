Descrição detalhada

A classe Sculptor representa um escultor 3D.

O escultor é representado por uma matriz tridimensional de voxels, onde cada voxel possui uma cor RGBA.

É possível realizar operações de desenho e corte na escultura, como colocar um voxel, cortar um voxel, colocar uma caixa, cortar uma caixa, colocar uma esfera, cortar uma esfera, colocar uma elipsoide e cortar uma elipsoide.

Também é possível salvar a escultura em um arquivo OFF.
